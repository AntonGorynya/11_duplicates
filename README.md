# Anti-Duplicator

This script remove same files in input directory and subdirectories

# Quickstart

You can download and run it directly through console

Example of script launch on Linux, Python 3.5:

```#!bash

duplicates.py ./
remove  ./1\2\3\2.txt
remove  ./1\2\3.txt
remove  ./1\2\4.txt


```
